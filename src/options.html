Plex token:
Learn how to find your token

Plex machine ID:
In Plex Web, click on a random movie. In the URL bar, copy the part between /server/ and /details/. Example: 992dggg6f248454c1d83de10a56c416570a81c84

Plex url root:
In Plex Web, open the DevTools (F12). You should see one or two messages about "Testing connection". Use the one with your external IP. Copy and paste the URL after the message here. Example: https://85-123-211-72.qe23e81fd3dcdda73ff82cxxxb446b0ff.plex.direct:32400

Plex library id:
In Plex Web, go to the library you want the extension to use. In the URL bar, copy and paste the number after /section/. Example: 1

CouchPotato URL (optional):
Example: https://example.com/couchpotato or http://192.168.1.100:5050 (no trailing slash!)

CouchPotato API key (optional):
In CouchPotato, go to Settings, click on "Show advanced" and copy paste the value next to "Api Key". Example: aa756d33242f6g8ffbca2b3963586f21

CouchPotato basic auth username (optional):
If you password-protected your CouchPotato install, enter the username here.

CouchPotato basic auth password (optional):
If you password-protected your CouchPotato install, enter the password here.
